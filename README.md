# Albania360

Albania360 è una web app responsive mobile-first per scoprire il turismo in Albania con un'estetica premium, cinematografica e ispirata alle piattaforme streaming.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS

## Pagine implementate

- `/` — Home con hero fullscreen, rail orizzontali Top 10, gemma nascosta e itinerari.
- `/esplora` — Catalogo filtrabile per categoria.
- `/contenuti/[slug]` — Pagina dettaglio riutilizzabile per luoghi, esperienze, tradizioni, gastronomia e itinerari.

## Comandi

```bash
npm install
npm run dev
npm run build
npm run typecheck
```
